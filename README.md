# DevForge AI

DevForge AI is a full-stack developer workspace for planning projects, tracking bugs, preserving knowledge, and generating AI-assisted documentation.

## Features
- Authenticated workspace with Supabase Auth
- Projects and bug vault modules
- Developer memory and knowledge base
- AI workspace for README, docs, commit messages, bug explanations, release notes, and summaries
- Responsive dashboard, analytics, profile, settings, notifications, and search
- Supabase Storage and Resend email integration

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, Framer Motion, Recharts
- Backend: Express, Supabase, Gemini, Resend

## Getting Started
1. Install dependencies for the client and server.
2. Configure environment variables from the example files.
3. Start the client and server.

## Deployment
- Frontend: Vercel
- Backend: Railway
